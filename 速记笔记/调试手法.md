# 1. 暴力调试法
这种方式是需要的思考较多，但效率也较低，一般简单的错误可以用，或者其他方法都试过了但是没找到错误。

大致分为三种类型：
1. 内存调试法：没用过
2. 打印语句调试：常用
3. 自动化调试工具进行调试：共同特点是提供断点机制，可以查看运行时的数据

# 2. 归纳调试法
归纳法从线索（即出错的症状，可能是1个或多个测试案例的结果）出发，寻找线索之间的联系，其过程如下：
1. 确定相关数据
2. 组织数据
3. 做出假设
4. 证明假设
5. 解决问题

# 3. 演绎调试法
演绎法从一些普遍的理论或前提出发，使用排除和精炼的过程，得到正确结论（找到错误位置）
