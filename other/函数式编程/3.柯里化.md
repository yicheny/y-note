# 定义
柯里化：只传递给函数一部分参数来调用它，让它返回一个函数去处理剩下的参数。
> 只传递给函数一部分参数也叫做局部调用(partial application)，能够大量减少样板代码。

```js
const add = x => y => x+y;

add(1,3);//4
```

# 预加载
利用柯里化，函数具有一种名为**预加载**的能力，通过传递一到两个参数调用函数，可以得到一个记住这些参数的新函数。<br/>
以刚刚的`add`函数示例：
```js
const addTen = add(10);

addTen(3);//13
addTen(5);//15
addTen(10);//20
```
